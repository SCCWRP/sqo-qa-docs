Benthic

The SQOUnified package provides several functions for analyzing benthic data:

- Benthic Response Index Function (BRI) - Relative Benthic Index Function (RBI) - Index of Biotic Integrity Function (IBI) - RIVPACS Function (RIVPACS) - Multivariate AMBI (MAMBI) - Benthic Integrated SQO Score Function (benthic.sqo)

Below are examples of how to use each function, along with how to view their outputs. We will use the example dataset, benthic_sampledata.

Benthic sample dataset:

        
print(head(benthic_sampledata, 10))

Here is an example of how to use the BRI function (using the sample data)

        
# Benthic Response Index
bri_scores <- BRI(benthic_sampledata)

Each function provides an output tailored to the benthic analysis it conducts, including various indices and biotic integrity scores. Together, these functions contribute to a comprehensive assessment of benthic data within the SQOUnified framework.
