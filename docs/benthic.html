Benthic

The SQOUnified package provides several functions for analyzing benthic data:

- Benthic Response Index Function (BRI) - Relative Benthic Index Function (RBI) - Index of Biotic Integrity Function (IBI) - RIVPACS Function (RIVPACS) - Multivariate AMBI (MAMBI) - Benthic Integrated SQO Score Function (benthic.sqo)

Below are examples of how to use each function, along with how to view their outputs. We will use the example dataset, benthic_sampledata.

Benthic sample dataset:

        
print(head(benthic_sampledata, 10))

Benthic Response Index Function

Here is an example of how to use the BRI function (using the sample data)

        
# Benthic Response Index
bri_scores <- BRI(benthic_sampledata)

# print the result
print(head(bri_scores, 7))

Relative Benthic Index Function

Here is an example of how to use the RBI function (using the sample data)

        
# Relative Benthic Index
rbi_scores <- RBI(benthic_sampledata)

# print the result
print(head(rbi_scores, 7))

Index of Biotic Integrity Function

Here is an example of how to use the IBI function (using the sample data)

        
# Index of Biotic Integrity
ibi_scores <- IBI(benthic_sampledata)

# print the result
print(head(ibi_scores, 7))

RIVPACS Function

Here is an example of how to use the RIVPACS function (using the sample data)

        
# RIVPACS
rivpacs_scores <- RIVPACS(benthic_sampledata)

# print the result
print(head(rivpacs_scores, 7))

Multivariate AMBI Function

Here is an example of how to use the MAMBI function (using the sample data)

        
# MAMBI
mambi_scores <- MAMBI(benthic_sampledata)

# print the result
print(head(mambi_scores, 7))

Benthic Integrated SQO Scores Function

Here is an example of how to use the benthic.sqo function (using the sample data)

        
# Integrated Benthic SQO Function
benthic_integrated_scores <- benthic.sqo(benthic_sampledata)

# print the resulting dataframe
print(head(benthic_integrated_scores, 6))

Each function provides an output tailored to the benthic analysis it conducts, including various indices and biotic integrity scores. Together, these functions contribute to a comprehensive assessment of benthic data within the SQOUnified framework.
