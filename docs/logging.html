# Turning on Verbose Logging

By default, the SQOUnified package functions do not log any information. However, you can turn on verbose logging to see more information about the calculations being performed. To do this, you can set the verbose parameter to TRUE when calling a function. Here's an example: 


# Load the SQOUnified package
library(SQOUnified)

# Turn on verbose logging for the tox.sqo function
tox_scores <- tox.sqo(tox_data, verbose = TRUE)

By default, this will create a folder called "logs" in the working directory in which you ran the code

Note: You can see what your current working directory is by running getwd() in the R console
