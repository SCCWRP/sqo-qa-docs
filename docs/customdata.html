Using Your Own Data with SQOUnified Functions

This guide will help you use your own data with the SQOUnified package's functions for different Lines of Evidence (LOE), including tox.sqo, benthic.sqo, chemdata_prep, and chem.sqo. Follow the steps below to load your data and apply these functions.

Loading Your Data

First, you need to load your data into R. This example shows how to read data from CSV and Excel files. Ensure that your data files are correctly formatted with the appropriate column names required by each function.

        
# Load required libraries
library(readxl)   # For reading Excel files

# Reading data from CSV
tox_data <- read.csv("path/to/your/tox_data.csv")
benthic_data <- read.csv("path/to/your/benthic_data.csv")
chem_data <- read.csv("path/to/your/chem_data.csv")

# Reading data from Excel
tox_data <- read_excel("path/to/your/tox_data.xlsx", sheet = "Sheet1")
benthic_data <- read_excel("path/to/your/benthic_data.xlsx", sheet = "Sheet1")
chem_data <- read_excel("path/to/your/chem_data.xlsx", sheet = "Sheet1")

Once your data is loaded, you can use the available functions from the package to calculate any scores you need


# Load the SQOUnified package
library(SQOUnified)
    
# Calculate toxicity scores using tox.sqo
tox_scores <- tox.sqo(tox_data)
print(head(tox_scores))


# Calculate chem scores using chem.sqo
chem_scores <- chem.sqo(chem_data)
print(head(chem_scores))


# Calculate benthic scores using benthic.sqo
benthic_scores <- benthic.sqo(benthic_data)
print(head(benthic_scores))


# Calculate all scores using SQOUnified
all.sqo <- SQOUnified(
    benthic = benthic_data,
    chem = chem_data,
    tox = tox_data
)
print(head(all.sqo))

Note: Your custom input data should be formatted in the same fashion as the sample input data, as shown on the Home Page
